# SubTracker

A web-based application that helps users track all their subscriptions,
see monthly and yearly costs, and receive reminders before renewals.

## Motivation
Many people lose money on forgotten subscriptions. This project aims to
make subscription spending transparent and manageable.

## Planned Features
- User authentication
- Add and manage subscriptions
- Monthly and yearly cost overview
- Upcoming renewal reminders
- Email notifications

## Tech Stack (Planned)
- Frontend: React
- Backend: FastAPI
- Database: PostgreSQL



## MVP Features
- User authentication (register and login)
- Add and manage subscriptions
- Monthly and yearly cost overview
- Upcoming renewal reminders
- Email notifications

## User Flow
1. User creates an account or logs in
2. User adds subscriptions with price and renewal date
3. User views a dashboard with total costs
4. User receives reminders before renewals
